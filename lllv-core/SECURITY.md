# Security Policy

## Supported Versions
- Main branch e último release publicado.

## Reporting a Vulnerability
Envie e-mail para **security@logline.world** com detalhes, PoC e impacto estimado.
Responderemos em até **7 dias** úteis, com triagem e plano de correção.

## Scope
- Integridade/Autenticidade de cápsulas (CID / assinatura)
- Criptografia (ChaCha20-Poly1305, AAD binding)
- Reprodutibilidade (header/binário)
